# レシピ16: Git操作とブランチ戦略を自動化する

Claude Codeの組み込みGit機能を活用し、ブランチ作成からPR提出までの一連のフローを自動化します。

## 使い方

1. Claude Codeに直接Git操作を依頼します。

```bash
> feature/user-profileブランチを作成して切り替えてください

> 現在の変更をコミットしてください。コミットメッセージは変更内容から自動生成してください

> 変更をリモートにプッシュして、PRを作成してください
```

2. 効率的なブランチ戦略のために、`CLAUDE.md`にGitワークフローのルールを記述します。

3. パーミッション設定でGit操作を効率化します。

4. 一連のワークフローを1つの指示で実行します。

```bash
> Issue #42を読んで、以下のフローで作業してください:
  1. feature/issue-42 ブランチを作成
  2. Issueの内容に基づいて実装
  3. テストを実行して全てパスすることを確認
  4. コミットしてPRを作成
```

5. `--from-pr`フラグでPRに紐づいたセッションを開始します。

```bash
# PRのコンテキストを引き継いでセッション開始
claude --from-pr 42
```

## 設定ファイル

- `settings.json`: Git操作のパーミッション設定（allow + deny）
- `CLAUDE.md`: Gitワークフローのルール例
