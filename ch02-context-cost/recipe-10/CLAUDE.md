# Project Name

## Overview

## Coding Standards
- TypeScript strict mode enabled
- Use Zod for input validation
- All endpoints must have OpenAPI annotations

## Commands
- `npm run dev` - Start dev server
- `npm test` - Run tests
- `npm run lint` - ESLint check
