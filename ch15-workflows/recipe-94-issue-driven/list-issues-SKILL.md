# Issue一覧の取得

GitHub MCPを使用して以下を実行してください。

1. 現在のリポジトリのオープンIssueを取得する
2. 以下のフォーマットで一覧表示する:

| # | タイトル | ラベル | アサイン | 作成日 |
|---|---------|--------|---------|--------|

3. ラベルに基づいて優先度を判定する:
   - `priority: high` → 最優先
   - `bug` → 高優先
   - `enhancement` → 通常
   - `good first issue` → 低優先

4. 優先度順にソートして表示する
