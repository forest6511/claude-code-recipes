# アクセシビリティ監査プロンプト

以下のプロンプトをClaude Codeに入力して、WCAG 2.1準拠チェックを実行します。

## 基本チェック

```text
http://localhost:3000 のトップページのアクセシビリティを
WCAG 2.1 Level AAの基準でチェックして。
問題があれば優先度付きでレポートして。
```

## 複数ページの網羅チェック

```text
サイト内の主要ページ（トップ、検索、商品詳細、カート、お問い合わせ）を
順番にチェックして、結果を1つのレポートにまとめて。
```

## キーボード操作の検証

```text
トップページのTabキーによるフォーカス順序を確認して。
フォーカスが見えなくなる箇所やキーボードトラップがないかチェックして。
```

## レポート出力形式の指定

```text
アクセシビリティの問題をマークダウン形式でレポートして。
Critical/Major/Minorの3段階で分類し、
各問題にWCAGの該当基準番号と修正方法を含めて。
```
